# Changelog

Todas as mudanças relevantes deste projeto serão documentadas aqui.

## [0.2.0] - 2026-02-04
### Breaking changes
- Nova arquitetura com conjuntos `outlined` e `filled`
- Imports modernos via `@iconis/icons/outlined`
- Compatibilidade com projetos antigos via namespace (`outlined.starRegular`)
- Renomeação dos ícones (remoção do sufixo `Icon`)

### Added
- Suporte a dois modos de import (namespace e subpath)
- Novo icone: Star
